# website-review-cancel-send-orders
Interactive website to add an order-review it and then cancel or send it.
The home page allows user to add information such as shipping address, payment method and shipment method
Using four interactive buttons ob the top, you can review order which then will be reflected in the textarea
After which you can either add it or cancel it. Then if it is correct you can send order which will take you to the next page which will display order
in the form of a table.
This website creates Shipment class and then adds orders in a table.
